### Sims Data

This directory is to store the direct output data from the [NQCDynamics.jl](https://github.com/NQCD/NQCDynamics.jl) simulations.

To process the NQCD raw data into analysis-ready data, please refer to the scripts in the [scripts/data_engineering](https://github.com/NQCD/HonGeAnalysis/tree/main/scripts/data_engineering) directory.
